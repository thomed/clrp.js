clrp.js

A standalone library for color picker inputs

clrp.js is designed as a "drop in and forget about it" library.

Once it is added to your page, you can set any <input> tags to be color pickers by simply adding the clrp class to the element:

<input class="clrp">

clrp inputs will trigger regular onchange events so you can react to the events just like normal inputs: 

<p id="text-display"></p>
<input class="clrp" id="my-clrp">
<script>
    var ptag = document.getElementById("text-display");
    var myClrp = document.getElementById("my-clrp");

    myClrp.addEventListener('change', function() {
        ptag.innerText = "Color: " + myClrp.value;
        ptag.style.color = myClrp.value;
    });
</script>
